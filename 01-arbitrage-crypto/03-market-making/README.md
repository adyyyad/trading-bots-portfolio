
# Market Making Bot

## Overview
Dynamic market making bot that provides liquidity and captures spread.

## Supported Markets
- Crypto (BTC/USDT, ETH/USDT)
- ETFs (SPY, QQQ)

## Key Features
- Dynamic spread adjustment based on volatility
- Inventory management to avoid directional risk
- Real-time order book analysis

## Performance
| Metric | Value |
|--------|-------|
| Daily Return (on liquidity) | 0.15% |
| Max Drawdown | 0.3% |
| Sharpe Ratio | 2.8 |

## Risk Management
- Inventory limits (max 20% of capital in one direction)
- Kill-switch for abnormal volatility
- Automatic order cancellation on drawdown
