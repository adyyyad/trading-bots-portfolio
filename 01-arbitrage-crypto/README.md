
# Trend Following Forex Bot

## Overview
Multi-timeframe trend following system for forex markets.

## Supported Pairs
- EUR/USD
- GBP/USD
- USD/JPY

## Key Features
- Trend detection using adaptive moving averages
- Multi-timeframe confirmation (15m, 1h, 4h)
- Volatility-adjusted position sizing

## Performance
| Metric | Value |
|--------|-------|
| Monthly Return | 1.8% |
| Max Drawdown | 2.1% |
| Sharpe Ratio | 1.9 |

## Risk Management
- Kill-switch
- ATR-based stop-loss
- Daily loss limits
