# Algorithmic Trading Bot Portfolio

## About me
I build production-ready algorithmic trading systems for crypto, forex, and equities.

**What I deliver:**
- Low-latency arbitrage bots (crypto exchanges)
- Adaptive trend-following systems (forex, stocks)
- Dynamic market making with inventory management
- **Neural network hybrids** (LSTM + Reinforcement Learning) for predictive execution

**What makes me different:**  
Most developers specialize in one strategy. I build **hybrid systems** where arbitrage, market making, and neural predictions work together. The neural layer learns from market conditions and adjusts strategy parameters in real-time.
